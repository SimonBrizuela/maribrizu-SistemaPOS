⏳ 1. INTEGRACIÓN CON GOOGLE SHEETS (Prioritario)
   ════════════════════════════════════════════════════════════════════════════
   
   📊 Objetivo: Que el dueño vea ventas en tiempo real desde cualquier lugar
   
   Pasos necesarios:
   ─────────────────────────────────────────────────────────────────────────
   
   A) CONFIGURACIÓN INICIAL
      □ Crear proyecto en Google Cloud Console
      □ Habilitar Google Sheets API
      □ Crear credenciales (Service Account)
      □ Descargar archivo JSON de credenciales
      □ Crear Google Sheet y compartir con service account email
   
   B) IMPLEMENTACIÓN EN EL SISTEMA
      □ Crear módulo: pos_system/sync/google_sync.py
      □ Implementar sincronización automática cada 5-10 minutos
      □ Sincronizar:
         • Ventas (tiempo real)
         • Productos (bidireccional)
         • Cierres de caja
         • Estado de stock
   
   C) ESTRUCTURA DEL GOOGLE SHEET
      Hojas necesarias:
      • "Ventas" - Registro completo de ventas
      • "Productos" - Lista maestra sincronizada
      • "Caja" - Aperturas y cierres
      • "Dashboard" - Resumen automático con fórmulas
      • "Config" - Configuración de precios/productos
   
   D) PANEL DE CONFIGURACIÓN EN EL POS
      □ Ventana de configuración de sincronización
      □ Estado de conexión (conectado/desconectado)
      □ Botón "Sincronizar ahora"
      □ Elegir qué datos sincronizar
      □ Ver última sincronización exitosa

                            PASOS PARA GOOGLE SHEETS
═══════════════════════════════════════════════════════════════════════════════

📋 GUÍA DETALLADA:

1. IR A GOOGLE CLOUD CONSOLE
   https://console.cloud.google.com/
   
2. CREAR NUEVO PROYECTO
   • Nombre: "POS Sistema" (o el que quieras)
   • Activar el proyecto

3. HABILITAR GOOGLE SHEETS API
   • Buscar "Google Sheets API"
   • Hacer clic en "Habilitar"

4. CREAR CREDENCIALES
   • Ir a "Credenciales"
   • Crear credenciales → Cuenta de servicio
   • Nombre: "pos-sync-service"
   • Crear y continuar
   • Descargar el JSON

5. CONFIGURAR GOOGLE SHEET
   • Crear nuevo Google Sheet
   • Nombre: "POS [Tu Negocio]"
   • Compartir con el email del service account
   • Dar permisos de "Editor"

6. GUARDAR CREDENCIALES
   • Copiar el archivo JSON a: pos_system/config/credentials.json
   • NO compartir este archivo (es privado)
