CÓMO EJECUTAR EL SISTEMA
=================================================================

1. Asegurarse de tener Python 3.8 o superior instalado

2. Instalar dependencias:
   py -m pip install -r requirements.txt

3. Ejecutar la aplicación:
   py main.py

4. La aplicación abrirá con:
   - Dashboard con estadísticas
   - Sistema de ventas
   - Gestión de productos
   - Control de caja

=================================================================
FLUJO DE TRABAJO RECOMENDADO
=================================================================

1. ABRIR CAJA (🏦 Caja)
   - Ir a la pestaña de Caja
   - Hacer clic en "Abrir Caja"
   - Ingresar monto inicial
   - Confirmar apertura

2. AGREGAR PRODUCTOS (🛍️ Productos)
   - Ir a la pestaña de Productos
   - Hacer clic en "Nuevo Producto"
   - Llenar información y agregar imagen
   - Guardar

3. REALIZAR VENTAS (💰 Ventas)
   - Ir a la pestaña de Ventas
   - Buscar productos o escanear código de barras
   - Agregar al carrito
   - Seleccionar tipo de pago
   - Completar venta (genera ticket PDF)

4. HACER RETIROS (🏦 Caja)
   - Ir a la pestaña de Caja
   - Hacer clic en "Retiro de Efectivo"
   - Ingresar monto y motivo
   - Confirmar (genera ticket PDF)

5. CERRAR CAJA (🏦 Caja)
   - Al final del día, ir a Caja
   - Hacer clic en "Cerrar Caja"
   - Contar efectivo real
   - Ingresar monto contado
   - Confirmar cierre (genera reporte PDF)

6. VER ESTADÍSTICAS (📊 Dashboard)
   - Dashboard se actualiza automáticamente
   - Ver ventas del día
   - Productos más vendidos
   - Gráficos por hora
